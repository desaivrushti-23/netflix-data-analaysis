# netflix-data-analaysis
# 🎬 Netflix Data Analysis using Python & Power BI

## 📌 Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python for data cleaning, exploratory data analysis (EDA), and visualization, followed by an interactive Power BI dashboard for business insights.

The objective is to understand Netflix's content library by identifying trends in content type, countries, genres, ratings, release years, and directors through meaningful visualizations and interactive dashboards.

# 💡 Key Business Insights

- Movies make up the majority of Netflix's content.
- The United States contributes the largest number of titles.
- TV-MA is one of the most common content ratings.
- International Movies is among the most popular genres.
- Netflix experienced rapid content growth after 2016.
- A small number of directors contributed multiple titles.

# 🚀 How to Run the Project

1. Clone this repository

```bash
git clone https://github.com/yourusername/Netflix-Data-Analysis.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run **Netflix_Data_Analysis.ipynb**

5. Open **Netflix_Dashboard.pbix** using Power BI Desktop.
